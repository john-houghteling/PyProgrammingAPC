# (Python) Programming for APC

This repository contains material for the course on Python programming and related amenities.

The structure of the directory is as follows:

- [slides](slides) contains the slides presented at odd lecture
- [hands-on](hands-on) contains the exercises done during even lectures
- [misc](misc) contains additional useful material

The structure of the lectures are as follows:

- Lecture 1: An introduction to the Shell and to Git
- Hands-on 1: Let's learn some basics on the field, generating batches and scripts, managing some files, generating a git-hub repo
- Lecture 2: Python 101
- Hands-on 2: In which we start working with python and sampling
- Lecture 3: Python OOP and Packaging
- Hands-on 3: we would like to have a re-usable package for MCMC
- Lecture 4: Parallelism, Architectures and Python
- Hands-on 4: let's wrap everything up by parallelising what we got up to here
